# Executar em dev

- `npm i`
- Criar '.env.dev'
- Criar '.docker-compose.yml'
- `npm run db:dev`
- `npm run wsl:dev`

# Executar em prod

- `npm i`
- Criar '.env.prod'
- Criar '.docker-compose.yml'
- `npm run db:prod`
- `npm run wsl:dev`
